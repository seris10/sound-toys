Karplus–Strong Strings

The simplest physical-modeling synthesis there is: fill a delay line with noise, then repeatedly average each sample with its neighbour. Each round of averaging is a low-pass filter; the loop length sets the pitch. Tap a string anywhere to pluck it — closer to the edge gives a brighter tone, like plucking near a guitar bridge. Each string glows in its own colour, mapped to its pitch, so the set forms a rainbow ladder.
